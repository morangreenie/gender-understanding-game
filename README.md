# GenderQuest

GenderQuest is a bright, child-friendly browser game that teaches four ideas:

1. body and sex traits;
2. gender identity;
3. gender expression; and
4. gender nonconformity, explained as interests, traits, and being yourself when other people expect something different.

Poison, Love, and Ghoul guide the player through four short lessons, six character stories, three quiz rounds, and a printable certificate.

## Play

Open the hosted game in Chrome:

**https://morangreenie.github.io/gender-understanding-game/**

The app works on phones, tablets, and computers. There are no accounts, advertisements, analytics, cookies, or external services. The player's first name and progress remain in that browser's local storage.

## Clinical-use note

The game is an educational conversation aid, not an assessment and not a way to decide a child's identity. It tells children that they do not need to label themselves, that it is okay not to know, and that hobbies and feelings do not inherently belong to one gender.
